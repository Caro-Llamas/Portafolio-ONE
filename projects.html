Videojuego para prevenir la violencia de género

    Este videojuego fue desarrollado con el motor Unity 3D, utilizando programación orientada a eventos con C#. Los modelos 3D fueron descargados gratuitamente de Unity Assets Store y los sprites 2D fueron diseñados por nosotras mismas. Para alcanzar un proceso de desarrollo de software adecuado, mi equipo y yo utilizamos la metodología SCRUM y aplicamos prácticas de QA Testing manuales.

    He desarrollado esta página web interactiva como resultado de concluir parte del programa Oracle Next Education. Permite introducir un mensaje y reacomodar los caracteres para que el mensaje introducido sea complicado de leer. De igual forma se puede volver al mensaje original presionando el botón desencriptar.
